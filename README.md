## Git
###  이름 설정 (GitHub 계정명과 동일하게 권장)
git config --global user.name "홍길동"

###  이메일 설정 (GitHub 가입 이메일과 동일하게)
git config --global user.email "gildong@example.com"

###  설정 확인
git config --list

###  모든 파일 스테이징
git add .

###  커밋 (변경사항 저장)
git commit -m "Initial STM32F103 project setup"

###  GitHub에 업로드
git push origin main
